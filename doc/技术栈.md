# 后端技术栈
技术类别	推荐方案	说明
API服务	Spring Boot 3 / FastAPI	支持 RESTful + GraphQL 双协议，Java/Python 可选
图数据库	Neo4j	存储本体对象类型间的关联关系与图遍历查询
关系数据库	PostgreSQL	存储对象实例数据、用户信息、审计日志
版本控制	Event Sourcing 模式	以事件流记录本体所有变更，支持时间回溯与版本重放
权限引擎	RBAC + ABAC 混合模型	角色权限+属性策略，支持对象类型级、属性级细粒度控制
消息队列	Apache Kafka	数据同步事件流，Action 执行事件推送
搜索引擎	Elasticsearch	对象实例全文检索、本体名称模糊搜索

# 前端技术栈
技术类别	推荐方案	说明
UI框架	React 18 + TypeScript	主流企业级框架，类型安全，生态完善
组件库	Ant Design Pro（深度定制）	企业级UI组件，配合设计Token覆盖保持品牌一致性
状态管理	Zustand + React Query	轻量全局状态+服务端数据缓存与同步
图谱可视化	AntV G6 / Cytoscape.js	支持大规模图数据渲染，交互性强
代码编辑器	Monaco Editor	用于规则逻辑、JSON 编辑场景
数据图表	ECharts / Recharts	仪表盘统计图表，雷达图、饼图、折线图
